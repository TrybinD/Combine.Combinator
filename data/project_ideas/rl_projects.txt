Оптимизация управления роботом в реальном времени: Создание системы для оптимизации управления роботом в реальном времени, например, на основе методов мягкого Q-обучения или обучения с подкреплением на основе модели. Для этого данные можно собрать из реальных экспериментов с роботами, используя сенсоры и камеры.

Обучение автономной навигации для беспилотных автомобилей: Разработка алгоритмов обучения с подкреплением для автоматического управления беспилотными автомобилями. Данные для этого проекта можно получить из симуляций движения по городской среде, таких как CARLA или примеров реальных данных от камер и LiDAR-сенсоров на автомобилях.

Данные для этого проекта можно собрать из различных симуляторов роботов с различными условиями.
